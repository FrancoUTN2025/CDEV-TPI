##  Scripts y Funcionalidad.

### main.js
Script principal de la funcionalidad del sistema, dentro de él esta la creacion del entorno con three.js, llamada a la iluminacion, creacion del motor del juego.



## Cómo usarlo

1. Agrega el archivo bosque6.glb a la carpeta src/models/Entornos


2. Ejecuta el archivo main.js
